# CV
Persönliche CV-Webseite, entwickelt im Rahmen der Projektarbeit.
Die Seite stellt meinen Lebenslauf dar.

## Login

Die Seite ist Passwortgeschützt.  
**Passwort: bbkklhl52**

Nach Eingabe des Passworts auf der Startseite (login.html) wird auf den CV weitergeleitet.

## Verwendete Technologien

- HTML (So semantisch wie möglich)
- CSS (Responsive)
- JavaScript (Dom, Events, Login Redirect)

Es wurden ausschliesslich Technologien verwendet, die im Kurs behandelt wurden.  
Keine externen Frameworks, KI wurde nur zur Hilfe genutzt um herauszufinden, wie es umgesetzt wird.  
Der gesamte Code kann erklärt werden und wurde von Hand geschrieben.

## Features

- Responsives Layout ( Desktop)
- Easter Egg: Klick auf das Profilfoto startet eine Canvas-Animation mit springenden Bällen und einer Begrüssungsfunktion mit Namenseingabe
- Login mit Passwortschutz (Weiterleitung auf passwortbenannte Seite)

## Projektstruktur

Die Entwicklung wurde in Branches durchgeführt.

- `main` – Zusammengeführter finaler Stand
- `CV` – Entwicklungsbranch
